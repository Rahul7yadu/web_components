# Component Libraries



### 1. Material-UI
- **Description**: A popular React component library implementing Google's Material Design.
- **Features**: Customizable components, theme support, responsive design.
- **Installation**: `npm install @mui/material @emotion/react @emotion/styled`
- **Usage Example**: [Material-UI Docs](https://mui.com/getting-started/installation/)
- **GitHub**: [mui/material-ui](https://github.com/mui/material-ui)
- **Website**: [mui.com](https://mui.com)

### - MUI X

        - MUI X is a collection of advanced React UI components for complex use cases. Use the native integration with Material UI or extend your design system.

        - Data Grid (complex table and Data view)
        - Date and Time Pickers
        - Charts
        - Tree View

## Daisy Ui 
- **Description** : daisyUI adds component class names to Tailwind CSS
- prebuilt themes like  "light",`
      "dark",
      "cupcake",
      "bumblebee",
      "emerald",
      "corporate",
      "synthwave",
      "retro", and more.[https://daisyui.com/docs/themes/]
- Number of components - 56
- Install - `npm i -D daisyui@latest`
- in tailwind config file `module.exports = {
    //...
    plugins: [
        require('daisyui'),
    ],
    }`
- Use - `<button class="btn">Button</button>`
